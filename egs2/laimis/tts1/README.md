# Laimis Text-to-Speech recipe

Written by Airenas Vaičiūnas @ VMU, Kaunas (2021)

## tts1 recipe

This is the recipe of Lithuanian single male speaker TTS corpus.

### Train

```bash
make train-fs2
## or
nohup make train-fs2 > v01.log &
```
